Hardware TOTP vs hardware FIDO2 key — key differences
Hardware TOTP token
Generates a 6-digit code on a screen
You type the code into the login form
No USB — no computer connection needed
Code can still be phished in real time
Works anywhere TOTP is supported
Hardware FIDO2 key (YubiKey)
Signs a cryptographic challenge silently
You just touch it — nothing to type
Requires USB or NFC connection
Phishing-proof — domain-bound
Only works where FIDO2 is supported
Battery warning: Hardware TOTP tokens have non-replaceable batteries. When the battery dies (typically 3–5 years), the token must be replaced entirely. Set a calendar reminder well in advance and always keep a spare token registered as backup MFA so you are never locked out when one expires.
